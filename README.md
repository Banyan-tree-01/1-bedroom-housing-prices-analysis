# 1-bedroom-housing-prices-analysis
The analysis uses Zillow's Home Value Index (ZHVI) for 1-bedroom single-family residences (smoothed, seasonally adjusted) across the top 100 US counties by size, from January 2000 to November 2025. After transposing the data to a time-series format and converting the index to datetime, missing values were handled using backward and forward fill.

This project was inspired by a kaggle analysis exploring Zillow housing data for price changes across diferent bedroom categories. This repository extends the exploration by focusing specifically on 1-bedroom housing price trends at the county level. 

Inspired By: Visualization of Zillow Economics Data by Niyamat Ullah
